Super Mario 64 Exploit Reference

Below is a compendium of the various exploits of Super Mario 64
Exploit Description
Backwards Long Jump
One of the most famous ways to achieve
Hyperspeed, backwards long jumping can be used to bypass most solid objects, with enough patience.
Hyperspeed walking
Under certain circumstances, Mario will try to move backward, but will be unable to, so the amount of backwards speed he has will build until he is moving too fast to be contained by a wall.
HOLP manipulation HOLP stands for “Held Object’s Last Position” and is used by the game to figure out where something should go when Mario drops it. You can trick the game into letting Mario drop things in places very far away from his hands, though.
Hat-in-hand glitch While not exactly useful on it’s own, the hat-in-hand glitch is useful for leading into other glitches, such as HOLP manipulation.
Pendulum Manipulation In Tick Tock Clock, if you set the level to random, you can control how much the pendulums swing by manipulating the random number generation.
